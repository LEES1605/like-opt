### 🔧 현재 해결 중인 문제

#### 1. 화면 스크롤 문제
- **문제**: 여전히 화면에 스크롤바가 보임
- **원인**: 컨텐츠가 화면 높이를 초과함
- **시도한 해결책**:
  - `position: fixed` 적용
  - `overflow: hidden` 설정
  - 요소 크기 축소 (제목, 헤더, 버튼 등)
  - Flexbox 레이아웃 적용
- **현재 상태**: 부분적 개선되었으나 완전 해결되지 않음

#### 2. 배경색상 문제
- **문제**: 배경이 흰색으로 표시되는 경우 발생
- **해결책**: `!important` 속성으로 CSS 우선순위 강화
- **현재 상태**: 대부분 해결됨
